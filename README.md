# Текстовый процессор (Go)

## Описание
CLI-программа на Go для обработки текста из файла.  
Читает входной файл, применяет правила и записывает результат в выходной файл.

---
## Возможности

- (hex) — перевод из 16-ричной в 10-ричную  
- (bin) — перевод из 2-ричной в 10-ричную  
- (up) — верхний регистр  
- (low) — нижний регистр  
- (cap) — первая буква заглавная  
- (up, N) / (low, N) / (cap, N) — применяет к N словам  

- исправляет пунктуацию  
- исправляет кавычки  
- заменяет `a` → `an` перед гласными и `h`  

---
## Запуск

```bash
go run . input.txt output.txt

Пример:
      it (cap) was the best (up) -> It was the BEST
